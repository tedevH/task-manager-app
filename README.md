# Full-Stack Task Manager

This project is a full-stack task management application built for the Webcraft Solutions Full-Stack Developer Intern assignment.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Filter tasks by All, Completed, and Pending
- Responsive React frontend
- Node.js backend with CRUD APIs
- Task storage in Postgres

## Tech Stack

- Frontend: React + Vite
- Backend: Node.js + Express
- Database: Postgres
